## 技术决策记录（ADR）

| 日期 | 决策 | 原因 |
|---|---|---|
| 初始 | 使用 Mock 用户替代认证 | 优先验证核心功能，认证作为独立 Sprint 4 接入 |
| 初始 | 时间线存储为 JSONB 而非关系表 | 时间线结构复杂且频繁整体更新，JSONB 更灵活 |
| 初始 | FFmpeg 处理在 Celery Worker 而非 API 进程 | 视频处理耗时长，必须异步避免阻塞 API |
| 初始 | 使用 Konva.js 渲染时间线 | DOM 元素数量过多时性能差，Canvas 方案可控 |
